## 2. Контекст и ограничения

**Предпосылки:**
- KonturTalk API работает только под admin key
- LLM должен иметь доступ только к записям пользователя
- MCP протокол выбран для стандартизации

**Риски:**

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Проблемы с JWT валидацией | Средняя | Высокое | Тестировать с разными токенами |
| Ограничения ktalk API | Высокая | Среднее | Использовать промежуточный сервер |
