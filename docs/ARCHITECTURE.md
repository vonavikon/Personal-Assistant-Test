# Архитектура Executive Assistant

## Обзор

Executive Assistant — шаблон репозитория для создания персонального AI-ассистента руководителя. Работает на базе VS Code + Claude Code.

## Структура

```
executive-assistant/
├── .claude/commands/     # Команды Claude Code (/init, /new-1-1, ...)
├── .claude/docs/         # Справочники (загружаются по необходимости)
├── presets/base/         # Шаблоны конфигурации (CLAUDE.md, AGENTS.md, ...)
├── templates/            # Шаблоны документов vault
├── scripts/              # Скрипты установки и настройки
└── docs/                 # Документация проекта
```

## Жизненный цикл

1. **Clone** — `git clone` репозитория
2. **Init** — `/init` запускает интервью и настройку
3. **Work** — используй команды `/new-1-1`, `/find-person`, ...
4. **Grow** — добавляй профили, проекты, решения

## Команда /init

Главная команда проекта. Выполняет:
1. Интервью (6 вопросов: компания, имя, должность, руководитель, подчинённые, контакты)
2. Создание структуры папок (Johnny Decimal)
3. Генерация конфигурации (CLAUDE.md, AGENTS.md, settings)
4. Создание профилей людей
5. Настройка инструментов (qmd, skills)

## Плейсхолдеры

Файлы `*.template` содержат плейсхолдеры `{{PLACEHOLDER}}`:

| Плейсхолдер | Описание |
|-------------|----------|
| `{{OWNER_NAME}}` | ФИО владельца |
| `{{OWNER_ID}}` | ID (транслит) |
| `{{OWNER_ROLE}}` | Должность |
| `{{COMPANY_NAME}}` | Компания |
| `{{VAULT_NAME}}` | Имя vault |
| `{{VAULT_PATH}}` | Путь к vault |
| `{{DOMAIN_DIR}}` | Домен (60_DOMAIN/...) |
| `{{DATE}}` | Дата настройки |
| `{{YEAR}}` | Текущий год |

## Инструменты

### qmd (опциональный)
Семантический поиск по vault. Установка: `npm install -g @tobilu/qmd`.
Без qmd — поиск через Grep/Glob (встроенные в Claude Code).

### Skills (опциональные)
- meeting-debrief — обработка транскриптов встреч
- correspondence-2 — деловая переписка
- meeting-prep — подготовка к встречам

## Субагенты (AGENTS.md)

5 специализированных агентов:
- **researcher** — поиск информации
- **meeting-processor** — обработка встреч
- **document-creator** — создание документов
- **writer** — написание текстов
- **analyst** — аналитика и рекомендации
