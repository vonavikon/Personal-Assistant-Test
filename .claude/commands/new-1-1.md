# Создание 1-1 записи

Создай запись 1-1 встречи с сотрудником.

## Алгоритм

### 1. Определи человека

Если имя не указано в аргументах — спроси: "С кем встреча?"

Найди ID:
- Если указан ID (латиница) → прямой поиск файла
- Если указано имя → `qmd_deep_search` или `Grep` по `10_PEOPLE/`

**Схема ID:** `{первая_буква_имени}{фамилия}` транслит → `amuratov`

### 2. Проверь папку

```
ls 10_PEOPLE/{id}/1-1/
```

Если папки нет — создай.

### 3. Загрузи шаблон

```
Read: 90_TEMPLATES/template_1-1.md
```

### 4. Заполни плейсхолдеры

| Плейсхолдер | Значение |
|-------------|----------|
| `{{person}}` | Полное имя |
| `{{date:YYYY-MM-DD}}` | Сегодняшняя дата |
| `{{date:DD MMMM YYYY}}` | Читаемый формат |

### 5. Создай файл

**Путь:** `10_PEOPLE/{id}/1-1/{YYYY-MM-DD}.md`

### 6. Подтверди

Сообщи путь к файлу и предложи заполнить секции.
