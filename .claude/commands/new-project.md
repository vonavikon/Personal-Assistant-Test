# Создание проекта

## Алгоритм

### 1. Собери информацию

Спроси:
- **Название проекта**
- **Тип:** стратегический / внутренний / кризисный
- **Приоритет:** критический / высокий / средний / низкий
- **Владелец** (по умолчанию — владелец vault)

### 2. Сгенерируй ID

Из названия: lowercase, транслит, дефисы вместо пробелов.
"Миграция на Kubernetes" → `migration-kubernetes`

### 3. Загрузи шаблон

```
Read: 90_TEMPLATES/template_project.md
```

### 4. Создай структуру

```bash
mkdir -p 30_PROJECTS/active/{id}
```

### 5. Заполни и запиши

**Путь:** `30_PROJECTS/active/{id}/{id}.md`

Заполни frontmatter: id, title, status, project_type, priority, owner, start_date.

### 6. Предложи следующие шаги

- Заполнить цель и критерии успеха
- Добавить команду проекта
- Определить этапы
