# Создание записи решения

## Алгоритм

### 1. Найди следующий номер

```
Glob: 40_DECISIONS/records/DEC-*.md
```

Следующий номер = max + 1, формат 4 цифры (DEC-0001).

### 2. Собери информацию

Запроси:
- **Контекст:** Какая проблема решается?
- **Решение:** Что решили?
- **Альтернативы:** Какие варианты рассматривались?
- **Последствия:** Риски и выгоды?

### 3. Загрузи шаблон

```
Read: 90_TEMPLATES/template_decision.md
```

### 4. Заполни frontmatter

```yaml
---
type: decision
id: DEC-{NNNN}
title: "{краткое название}"
date: YYYY-MM-DD
status: proposed
author: "{владелец vault из CLAUDE.md}"
---
```

### 5. Создай файл

**Путь:** `40_DECISIONS/records/DEC-{NNNN}.md`

### 6. Предложи следующие шаги

- Согласовать с заинтересованными сторонами
- Обновить статус на `accepted` после утверждения
