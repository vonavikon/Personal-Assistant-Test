# Фоновая обработка встречи

Запусти обработку транскрипта в фоновом режиме.

## Использование

`/bg-debrief path/to/transcript.txt`

## Алгоритм

1. Запусти отдельную сессию:
```bash
claude --remote "Используй /meeting-debrief для обработки: $ARGUMENTS"
```

2. Сообщи пользователю ID сессии для мониторинга через `/tasks`.

3. Результаты будут сохранены в vault, транскрипт архивирован.

## Когда использовать

- Большие транскрипты (>50KB)
- Когда не хочется тратить контекст основной сессии
